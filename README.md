# Yoruba Proverb Bot
A bot that posts a Yoruba proverb and its translation on Mastodon daily.

## Usage
### Set up
```
git clone
```

Rename `.env-example` to `.env` and paste in your own secret keys

### Running the bot
```
npm install
node yoruba-bot.js
```
